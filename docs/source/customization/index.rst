Customization
=================

MISO allows users to customize the source code for their specific needs. This section provides guidelines on how to modify and extend the MISO codebase.

MISOは、ユーザーが特定のニーズに合わせてソースコードをカスタマイズできるようにしています。本セクションでは、MISOコードベースの変更および拡張方法についてのガイドラインを提供します。
