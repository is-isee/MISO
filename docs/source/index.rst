MISO documentation
==================

MISO (Mhd ISee Open source code) is an open-source software for performing magnetohydrodynamics (MHD) simulations.

This software is developed mainly by the Integrated Studies Division/CIDAS, Institute for Space-Earth Environmental Research (ISEE), Nagoya University, Japan.

MISO (Mhd ISee Open source code)は、磁気流体力学(MHD)シミュレーションを実行するためのオープンソースソフトウェアです。

名古屋大学宇宙地球環境研究所(ISEE) 総合解析研究部/CIDASを中心に開発されています。
